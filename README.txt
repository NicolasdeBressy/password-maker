# Générateur de Mot de Passe

Ce projet est un générateur de mot de passe simple développé en HTML, CSS et JavaScript.

## Fonctionnalités

- Permet de générer des mots de passe aléatoires en fonction des critères sélectionnés.
- Possibilité de définir la longueur du mot de passe.
- Options pour inclure des lettres minuscules, majuscules, des chiffres et des caractères spéciaux.
- Bouton pour copier le mot de passe généré dans le presse-papiers.

## Comment utiliser

1. Ouvrez le fichier `index.html` dans un navigateur web.
2. Sélectionnez les critères souhaités pour le mot de passe.
3. Cliquez sur le bouton "Générer mot de passe" pour obtenir un mot de passe sécurisé.
4. Utilisez le bouton "Copier" pour copier le mot de passe dans le presse-papiers.
